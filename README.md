# 🧪 End-to-End E-Commerce Application Testing

## 📌 Project Overview
This project demonstrates end-to-end testing of an e-commerce web application using manual testing, automation testing, API testing, and database validation. The goal is to ensure the quality, reliability, and performance of the application.

---

## 🚀 Tech Stack & Tools
- **Automation Testing:** Selenium (Python) + Pytest  
- **API Testing:** Postman  
- **Database Testing:** MySQL (SQL Queries)  
- **Programming Language:** Python  
- **Test Management:** Test Cases & Bug Reports  

---

## 🔍 Features Tested
- User Login & Signup  
- Product Browsing  
- Add to Cart  
- Shopping Cart Validation  
- API Endpoints  
- Database Records  

---

## 🧪 Testing Types Covered
- Functional Testing  
- Regression Testing  
- API Testing  
- Database Testing

## 🔌 API Testing (Postman)

  - Status Codes (200, 201)  
  - Response Body  
  - JSON Structure  
  - Response Time

## 🗄️ Database Testing

## 🐞 Bug Reporting

  - Expected vs Actual results  
  - Severity & Priority
